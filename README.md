## Nedir?

**Astro Agent v30 Ultra**, basit bir yapay zeka değildir; devasa projeleri sıfırdan mimari eden, hataları sanal makineler ile test eden ve %100 matematiksel kesinlik ile çalışan bir Ajan Sistemidir. Tamamen yerel ortamınızda otonom olarak çalışır.

---

## Özellikler

- **Profesör Zekası:** Astro Agent, karmaşık problemleri sistem yaklaşımıyla çözer. Yeni formüller icat edebilir ve sıfır-hata matematiği sayesinde sayısal işlemlerde kesin sonuçlar verir.
- **Otonom Eylem:** Sürekli komut vermenize gerek kalmadan tarayıcıyı kontrol edebilir, terminalde araştırma yapabilir ve kodları bağımsız olarak derleyebilir.
- **Kırılmaz Hafıza:** Yüz binlerce satırlık veri işlemesinde sistemin çökmesini engelleyen özel "Compaction" motoruna sahiptir. Asla token şişmesi yaşanmaz.
- **MCP Bağlantısı:** Harici Veritabanları, Tarayıcı Uzantıları ve dış araçlarla direkt bağlantı kurabilen yeni nesil veri köprüsü.

---

## Kurulum

Sisteminizde Node.js ve Git kurulu ise şu adımları izleyin:

```bash
git clone https://github.com/vertexcorporation/Astro-Agent.git
cd Astro-Agent
npm install
npm link
```

> **Not:** Kurulum bittikten sonra asistan komutlarının işletim sisteminize işlemesi için açık olan tüm terminallerinizi kapatıp yeniden açın.

---

## Kullanım

Bilgisayarınızdaki herhangi bir projede terminalinizi açın ve asistanınızı uyandırın:

```bash
# Astro Agent'ı Başlat (En Hızlı Kısayol)
asg

# Alternatif komutlar
astro
astroagent
```
