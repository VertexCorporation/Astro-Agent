## Nedir?

**Astro Agent**, basit bir yapay zeka değildir. Devasa projeleri sıfırdan mimari eden, hataları arka planda izleyip düzelten, çoklu modeli Fusion Mode ile harmanlayan ve tarayıcıyı doğrudan kontrol eden tam otonom bir **AI Coding Agent**'tır. Tamamen yerel ortamınızda çalışır — hiçbir şeyi buluta göndermez.

---

## Özellikler

| Özellik | Açıklama |
|---|---|
| 🧠 **Profesör Zekası** | Karmaşık mimari problemleri sistem düşünmesiyle çözer. Sıfır-hata matematiği. |
| 🌐 **Web UI Studio** | `http://127.0.0.1:3135` üzerinde tam ekran, sohbet tabanlı modern arayüz. |
| 🔥 **Fusion Mode** | Düşünme modeli + kodlama modelini sırayla harmanlayan çift-model sistemi. |
| 👁️ **Astro Sentinel** | Dosya değişikliklerini arka planda izler, TypeScript hatalarını proaktif yakalar. |
| 🌉 **Browser Bridge** | Chrome extension üzerinden tarayıcıyı direkt kontrol eder. |
| 🔗 **MCP Bağlantısı** | Harici veritabanları ve araçlarla direkt köprü (Blender, Godot, Scratch vb.). |
| 📦 **Kırılmaz Hafıza** | Özel Compaction motoru — token şişmesi yaşanmaz, milyonlarca satır işler. |
| ⚡ **Otonom Eylem** | Tarayıcı, terminal, git, build — hepsini tek komutla yönetir. |
| 🎨 **Photo & Video Studio** | Yerleşik görsel ve video düzenleme araçları. |

## Kurulum

Node.js (18+) ve Git kurulu ise:

```bash
git clone https://github.com/vertexcorporation/Astro-Agent.git
cd Astro-Agent
npm install
cd packages/cli
npm run build
npm install -g .
```

> **Not:** Kurulum bittikten sonra yeni bir terminal açın; değişiklikler bu şekilde geçerli olur.

## Kullanım

Herhangi bir proje klasöründe terminali açıp çalıştırın:

```bash
# Kısa komut
asg

# Alternatif komutlar
astro
astroagent
mooncode
```

Uygulama başladıktan sonra tarayıcı otomatik açılır:

- **Web UI** → `http://127.0.0.1:3135`
- **Auth / MCP API** → `http://127.0.0.1:3131`
- **Browser Bridge** → `ws://127.0.0.1:3133`

---

## Web UI

Astro Agent'ın ana arayüzü **Studio Mode**'dur. Sol panel sohbet geçmişini, sağ panel sistem durumunu gösterir.

Hızlı aksiyonlar:
- `/build` — Projeyi derle
- `/test` — Test suite çalıştır
- `/ship` — Git branch, commit, push
- `/compact` — Context belleğini sıkıştır
- `/review` — Kod inceleme yap

---

## MCP

Yan panelden **MCP Manager**'ı açarak harici araçları bağlayın:

- **Blender** — 3D sahne kontrolü
- **Godot** — Oyun motoru entegrasyonu
- **Scratch MCP** — Özel araç köprüsü
- **Custom** — Kendi MCP sunucunuzu ekleyin
